# YuanQiKingnight - 元气骑士主题光标

这是一个基于游戏《元气骑士》中剑类武器元素设计的 Windows 系统光标主题包。

## 简介
- 主题来源：游戏《元气骑士》
- 适用系统：Windows

[image](https://github.com/user-attachments/assets/79b71733-b205-4420-a2ec-6212d00c2645)

## 使用方法
1. 下载光标文件
2. 解压到本地文件夹
3. 右键点击 .cur 文件进行安装
4. 在 Windows 鼠标设置中应用新的光标主题
